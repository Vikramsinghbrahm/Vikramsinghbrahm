# Vikram Singh Brahm

Software developer with a Master's in Applied Computer Science from Concordia University. I build backend platforms, distributed systems, and full-stack products with a focus on reliable APIs, operational visibility, and clean architecture.

My recent work spans distributed job orchestration, secure file transfer, document-centric workflow products, and experimentation platforms for graph analysis and applied machine learning. I am especially interested in backend, platform, and full-stack engineering roles where system design and product execution both matter.

## Focus Areas

- Backend and platform engineering in Go, Java, Python, and JavaScript/TypeScript
- Distributed systems design across durable queues, retries, idempotency, concurrency control, and service coordination
- Full-stack product delivery with Vue 3, React, Vite, and API-first backend design
- Observability and operations with OpenTelemetry, Prometheus, structured logging, Docker, and deployment automation
- Applied machine learning and computer vision using PyTorch, scikit-learn, OpenCV, and reproducible evaluation workflows

## Full-Stack Platforms

- `CareerLinkUp Platform`: designed a distributed hiring platform with Spring Boot microservices, Vue 3 + Vite, RabbitMQ domain events, JWT-secured gateway routing, database-per-service PostgreSQL persistence, correlation ID propagation, and CI quality gates
- `PacketFlow`: built a document operations workspace with collaborative annotations, checklist-driven review flows, signed access, immutable audit history, projection snapshots, export bundles, storage administration, and grounded ask-the-document workflows
- [VaultFlow Secure File Transfer](https://github.com/Vikramsinghbrahm/secure-file-transfer): built a secure file transfer control plane with Flask, Vue 3, JWT authentication, expiring share links, streamed upload and download paths, audit visibility, and rate limiting

## Backend and Distributed Systems

- `task-forge`: built a production-style distributed job queue in Go with a PostgreSQL-backed durable queue, leased worker claims, queue-aware concurrency, OpenTelemetry tracing, Prometheus metrics, structured JSON logs, DLQ replay and purge workflows, and a separate Vue 3 + Vite ops console
- [Selective Repeat Over UDP in Go](https://github.com/Vikramsinghbrahm/selective-repeat-implementation): implemented reliable HTTP-style file transfer over UDP using a selective-repeat sliding window, adaptive retransmission timing, authenticated session framing, and concurrent client-server sessions

## Research and Applied ML

- [Online Graph Coloring Analysis](https://github.com/Vikramsinghbrahm/Online-Graph-Coloring-Analysis): created an experimentation platform for comparing online graph coloring algorithms on randomly generated k-partite graphs, including reproducible benchmarks, graph visualization, and run-level performance metrics
- [Sign Language Classification](https://github.com/Vikramsinghbrahm/Sign-Language-Classification): contributed to a CNN-based research project comparing ResNet18, MobileNetV3, and ShuffleNetV2 across multiple sign language datasets; the best reported result on Dataset 2 reached 97.3% test accuracy
- [Retinal Vessel Extraction](https://github.com/Vikramsinghbrahm/Image-Processing_retinal-vessel-extraction): implemented an interpretable matched-filter and first-order derivative of Gaussian pipeline for retinal vessel segmentation, with dataset evaluation and reproducible accuracy, sensitivity, and specificity metrics

## Language and Foundations

- [Clojure Text Compression CLI](https://github.com/Vikramsinghbrahm/clojure-compress-decompress): built a dictionary-based text compression and decompression CLI in Clojure with resource-backed frequency lists, reversible file processing, and test coverage
