## Full-Stack Platforms

- [CareerLinkUp Platform](https://github.com/Vikramsinghbrahm/career-linkup): designed a distributed hiring platform with a Spring Boot microservices backend, a Vue 3 frontend on Vite, RabbitMQ-based domain events, database-per-service PostgreSQL persistence, JWT-secured gateway routing, correlation ID propagation, health probes, run scripts, and CI quality gates
- [PacketFlow](https://github.com/Vikramsinghbrahm/packet-flow): built a PDF-first insurance packet review workspace with collaborative annotations, checklist-driven decisions, signed session access, immutable audit events, projection snapshots, export bundles, storage admin operations, and a grounded Ask the Document RAG flow
- [VaultFlow Secure File Transfer](https://github.com/Vikramsinghbrahm/secure-file-transfer): built a secure file transfer control plane with JWT authentication, per-user storage isolation, expiring share links, audit logging, rate limiting, and streamed upload and download paths that avoid buffering entire files in memory

## Backend and Distributed Systems

- [Selective Repeat Over UDP in Go](https://github.com/Vikramsinghbrahm/selective-repeat-implementation): implemented reliable HTTP-style file transfer over UDP using a selective-repeat sliding window, adaptive retransmission timing, authenticated session framing, and concurrent client-server sessions
- [Distributed Leader Election Using Apache ZooKeeper](https://github.com/Vikramsinghbrahm/leader-election-algorithm-apache-zookeeper): implemented leader election and coordination across multiple application instances using ZooKeeper watchers
- [Music Quality Management Microservice](https://github.com/Vikramsinghbrahm/Music_Quality-Management-Microservice): built a Spring-based microservice for subscription-aware audio bitrate settings and configuration-driven service behavior

## Research and Applied ML

- [Online Graph Coloring Analysis](https://github.com/Vikramsinghbrahm/Online-Graph-Coloring-Analysis): created a full-stack experimentation platform for comparing online graph coloring algorithms on randomly generated k-partite graphs, including reproducible benchmarks, graph visualization, and run-level performance metrics
- [Sign Language Classification](https://github.com/Vikramsinghbrahm/Sign-Language-Classification): contributed to a CNN-based research project comparing ResNet18, MobileNetV3, and ShuffleNetV2 across multiple sign language datasets; the best reported result on Dataset 2 reached 97.3% test accuracy
- [Retinal Vessel Extraction](https://github.com/Vikramsinghbrahm/Image-Processing_retinal-vessel-extraction): implemented an interpretable matched-filter and first-order derivative of Gaussian pipeline for retinal vessel segmentation, with dataset evaluation and reproducible accuracy, sensitivity, and specificity metrics
